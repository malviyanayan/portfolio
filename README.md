# Nayan Malviya — Portfolio

Personal portfolio website built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:8080](http://localhost:8080)
